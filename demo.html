Chatbot Widget Demo

This page demonstrates the InfinitySkyAI Chatbot Widget. You can interact with the chat widget by clicking the chat icon in the bottom right corner.

Features of this widget:

  • Modern, clean design with smooth animations
  • Customizable colors, positioning, and branding
  • Typing indicators to show when responses are being generated
  • Responsive layout that works on all devices
  • Auto-resizing text input
  • Error handling and connectivity detection
  • Support for multiple message formats

How to Use on Your Website

Add the following code to your website's HTML:

<!-- Widget Configuration -->
<script>
    window.ChatWidgetConfig = {
        webhook: {
            url: 'YOUR_WEBHOOK_URL',
            route: 'general'
        },
        branding: {
            logo: 'YOUR_LOGO_URL',
            name: 'Your Company Name',
            welcomeText: 'Hi ðŸ‘‹, how can we help?',
            responseTimeText: 'We typically respond right away',
            poweredBy: {
                text: 'Powered by InfinitySkyAi',
                link: 'https://infinitysky.ai'
            }
        },
        style: {
            primaryColor: '#59b3ea',
            secondaryColor: '#59b3ea',
            position: 'right',
            backgroundColor: '#ffffff',
            fontColor: '#333333'
        }
    };
</script>
